*WaveMaker has released a new update version with bug fixes.*

## Improvement 

- For CAS security provider, you can view additional information of the logged-in user using securityService/getSecurityInfo API by creating a service variable for this API.

## Bug fixes

---

- When configuring the Live Phase, deployment to the cloud provider was failing. This is fixed.
- Displayed console error: "cannot read property ngForm of undefined" when you use chip widget inside the wizard widget. This is fixed. 
- Number widget validation was not updating when min/max value is dynamically set from other widgets. This is fixed. 
- There were performance issues observed when the compression property is enabled in the project.
- When adding app version, it was allowed to add only minor version upto 0-9 only. Now, the range is extended to 1-99. For example, 10.19.
