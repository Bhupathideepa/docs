MockingBird is a Platform which provides you APIs to mock imported APIs from WME Studio. MockingBird can be installed on K8s Cluster, it can be hosted and mnaged by an organization on their own K8s CLuster

You can setup MockingBird on any K8s Cluster

This documentation desribes the process for setting up MockingBird and integration with WaveMaker Enterprise.

## MockingBird Platform setup Process

MockingBird setup requires you to follow the steps below.

### Step-1: Understand prerequisites

Understand and procure the [Prerequisites](/learn/on-premise/extensions/prerequisites).

### Step-2: Choose an infrastructure provider

MockingBird deliverable is verified on following clusters

1. [GKE](https://cloud.google.com/learn/what-is-kubernetes)
2. [EKS](https://aws.amazon.com/eks/)
3. [AKS](https://learn.microsoft.com/en-us/azure/aks/)
4. [Minikube] (https://minikube.sigs.k8s.io/docs/start/)
